# Research Paper Tracker

A simple web app for students and researchers to log, organize, 
and rate academic papers they have read.

## Features

- Add papers with title, authors, topic, year, rating, and notes
- View all papers in a searchable, filterable table
- Dashboard showing total papers and breakdown by topic
- Deployed live via Vercel with Supabase backend

## Tech Stack

- **Frontend:** Next.js (generated with v0 by Vercel)
- **Database:** Supabase (PostgreSQL)
- **Deployment:** Vercel
- **AI Tools Used:** GitHub Copilot, v0 by Vercel

## Setup Instructions

1. Clone the repository
```bash
   git clone https://github.com/YOUR_USERNAME/research-paper-tracker.git
   cd research-paper-tracker
```
2. Install dependencies
```bash
   npm install
```
3. Create a `.env.local` file in the root with your Supabase credentials:
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key

4. Run locally
```bash
   npm run dev
```
